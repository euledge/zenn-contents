先日発表のGoogle DeepMindのAIエージェント「Antigravity」を使って、 **ノーコード（に近い感覚）** でコミュニティ用のAdvent Calendarアプリを開発した過程を紹介します。
AIエージェントを使った開発はしたことはあるものの、全くのノーコードでのバイブコーディングには懐疑的だったのですがどこまでできるのかのチャレンジをしてみました。

## 0. テーマの選定
12月になると毎年恒例の地域のITコミュニティ合同でのLT大会＆忘年会があります。
このイベントのLT枠はすぐに埋まってしまうほどの人気のイベントで、いつも出遅れ発表ができないのですが今年は諸般の事情で日程が合わず参加できなくしょぼんとしておりました。
そんな中でAntigravityの発表。ほかにもLT枠あふれてしまい発表できない人もいるだろうなと思ってふと思いついたのは、「じゃあこのコミュニティのアドベントカレンダー作ってみよう」
要件も明確だしバイブコーディングにちょうどいい題材かなとのひらめきで今回進めてみました。

Antigravityとの対話記録を振り返りながら記事を進めてみます。

※注 以下の対話の中でのAgentのふるまいは一部AGENTS.mdの指示を基に動いている個所もあります。

Agent Managerは、開発者が直接コードを書くというよりも、AIエージェントを指揮・監督し、複数のタスクやワークスペースを一元管理するための司令塔として機能します。

🎯 Agent Managerの **Workspace（ワークスペース）** 画面では、プロジェクト内のタスクや進行状況をまとめて管理し、ファイルやフォルダ構造を確認しながら、プロジェクト全体を把握できます。
今回の対話記録がここに残っています。

## 2. デザイン：「クリスマスっぽくして」

機能ができたら次は見た目です。ここでも「指示は日本語、実装はAI」のスタイルを貫きました。

> **User**: 「デザインをクリスマスっぽくしたい。ロゴに合わせて背景色を変えて、カレンダーは赤いボードにぶら下がってる感じで。」

この抽象的なオーダーに対し、Antigravityは：

*   **カラーパレット**: クリスマスグリーン (`hsl(148,49%,29%)`) や温かみのある背景色 (`#f6eedb`) を適用
*   **レイアウト**: グリッドデザインを調整し、カードがボードに並んでいるようなUIを構築
*   **装飾**: ヘッダーにミカン付きのツリーやプレゼント箱のイラスト（CSS/SVG）を配置

**💡 ここでワンポイント：AIの使い分け**
実は、ロゴやアイコン画像の生成に関しては、Geminiだけでなく **ChatGPT** も併用しました。今回のテーマである「イラストっぽいいい感じの画像」を作る上では、ChatGPTの方がイメージに近いものを出力してくれたためです。
得意分野に合わせて複数のAIを使い分けるのも、ノーコード開発を成功させるコツかもしれません。

「なんか違う」と思ったら「もっと文字を大きく」「ホバーした時の色を変えて」と言うだけで、CSSが勝手に書き換わっていく体験は圧巻でした。

**📸 スクショで「ここ！」と伝える**
言葉だけで伝えるのが難しいデザインの修正指示では、 **スクリーンショット** を一緒に渡す方法が非常に有効でした。
「この隙間を詰めて」「このアイコンをこっちに」といった指示も、画像を一枚貼るだけでAntigravityが文脈を完璧に理解し、補完してくれます。テキストと画像を組み合わせることで、コミュニケーションコストが劇的に下がりました。

## 4. 開発を終えて

今回の開発で私が書いたコードは、実質 **0行** です（AIが生成したコードの確認はしましたが）。

*   **要件定義** : 会話で実施
*   **コーディング** : Antigravity
*   **デバッグ** : エラーが出たらログを見せるだけで解決
*   **コミット** : Git操作もAIにお任せ
*   **ドキュメント** : ドキュメント作成や関連ドキュメントの更新もあいまいな指示でもお任せ

「Antigravity」は単なるコード生成ツールではなく、 **「文脈を理解して自律的に動くパートナー」** だと感じました。
これからの開発は、「どう書くか」ではなく「何を作りたいか」を語ることに集中できる時代になりそうです。

**成果物スペック**
*   **Frontend** : React, TypeScript, Tailwind CSS
*   **Backend** : Firebase (Firestore, Hosting)
*   **Features** :
    *   25日分のカレンダー表示
    *   記事登録・編集機能
    *   公開日制御（未来の記事はロック）
    *   レスポンシブデザイン
    *   クリスマス特別テーマ
